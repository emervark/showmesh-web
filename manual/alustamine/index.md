# Getting started

After this chapter you can install or start Showmesh, add media, build a cue
list, choose an output, and run a controlled first rehearsal.

## Recommended order

1. [Install and run Showmesh](/alustamine/paigaldamine).
2. Learn the [user interface](/alustamine/kasutajaliides).
3. Build [your first show](/alustamine/esimene-show).
4. Before a performance, use the [show preparation checklist](/toovood/show-ettevalmistamine).

## Before you begin

The current beta targets 64-bit Windows. Use a machine with suitable display
and audio outputs, enough GPU capacity for the intended media, and representative
test files. NDI output requires the NDI runtime.

::: tip Start in a window
Use a windowed program output while learning. The editor, program, and engine
log can then remain visible on one display.
:::
