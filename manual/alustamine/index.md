# Getting started

After this chapter you can start the engine and editor, add media, build a cue
list, choose an output, and run a controlled first rehearsal.

## Recommended order

1. [Install and run Showmesh](/alustamine/paigaldamine).
2. Learn the [user interface](/alustamine/kasutajaliides).
3. Build [your first show](/alustamine/esimene-show).
4. Before a real performance, use the [show preparation checklist](/toovood/show-ettevalmistamine).

## Before you begin

You need Windows 11, suitable audio and video outputs, and test media. The
development build also requires an engine and editor built from source. This
branch does not yet include an installer.

::: tip Start in a window
Use windowed program output while learning. The editor, program, and engine log
can then remain visible on one display.
:::
