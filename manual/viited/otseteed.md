# Keyboard shortcuts

## File and editing

| Shortcut | Command |
|---|---|
| <kbd>Ctrl</kbd>+<kbd>N</kbd> | New project |
| <kbd>Ctrl</kbd>+<kbd>O</kbd> | Open |
| <kbd>Ctrl</kbd>+<kbd>S</kbd> | Save |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd> | Save As |
| <kbd>Ctrl</kbd>+<kbd>Z</kbd> | Undo |
| <kbd>Ctrl</kbd>+<kbd>Y</kbd> or <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Z</kbd> | Redo |
| <kbd>Ctrl</kbd>+<kbd>C</kbd> / <kbd>Ctrl</kbd>+<kbd>V</kbd> | Copy / paste cue |
| <kbd>Ctrl</kbd>+<kbd>D</kbd> | Duplicate selected cue |
| <kbd>Delete</kbd> / <kbd>Backspace</kbd> | Delete selection in Edit mode |

## Transport and navigation

| Shortcut | Command |
|---|---|
| <kbd>Space</kbd> | GO outside text inputs |
| <kbd>Esc</kbd> | Close top layer; bare Esc starts PANIC |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Select and arm previous/next cue |
| <kbd>Shift</kbd>+<kbd>↑</kbd>/<kbd>↓</kbd> | Extend cue range |
| <kbd>Shift</kbd>+click | Select contiguous range |
| <kbd>Ctrl</kbd>+click | Toggle row without re-arming |
| <kbd>Ctrl</kbd>+<kbd>K</kbd> | Command palette |

## Program output

| Shortcut | Command |
|---|---|
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>D</kbd> | Disable program display |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>1…9</kbd> | Fullscreen on display 1…9 |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>0</kbd> | Windowed output on display 1 |

Editing and output changes are blocked in Show mode. The same shortcuts use
<kbd>Cmd</kbd> where supported.
