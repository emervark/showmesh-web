# Klaviatuuri otseteed

Windowsis kasuta <kbd>Ctrl</kbd>. Kood toetab samades kombinatsioonides ka
<kbd>Cmd</kbd> modifikaatorit.

## Fail ja muutmine

| Otsetee | Käsk | Show mode'is |
|---|---|---|
| <kbd>Ctrl</kbd>+<kbd>N</kbd> | Uus projekt | Lukustatud |
| <kbd>Ctrl</kbd>+<kbd>O</kbd> | Ava projekt | Lukustatud |
| <kbd>Ctrl</kbd>+<kbd>S</kbd> | Salvesta | Lukustatud |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd> | Salvesta nimega | Lukustatud |
| <kbd>Ctrl</kbd>+<kbd>Z</kbd> | Undo | Lukustatud |
| <kbd>Ctrl</kbd>+<kbd>Y</kbd> | Redo | Lukustatud |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Z</kbd> | Redo | Lukustatud |

## Transport ja navigeerimine

| Otsetee | Käsk |
|---|---|
| <kbd>Space</kbd> | GO, kui fookus pole sisestusväljal |
| <kbd>Esc</kbd> | Sulge käsupalett; muidu PANIC |
| <kbd>Delete</kbd> või <kbd>Backspace</kbd> | Kustuta valitud cue Edit-režiimis |
| <kbd>Ctrl</kbd>+<kbd>K</kbd> | Ava või sulge käsupalett |
| <kbd>Enter</kbd> | Kinnita numbriline või tekstiväli fookuse kaotamisega |

::: danger Escape on PANIC
Kui käsupalett pole avatud ja fookus pole sisestusväljal, on <kbd>Esc</kbd>
globaalne PANIC. Ära kasuta seda lihtsalt valiku tühistamiseks live-show ajal.
:::
