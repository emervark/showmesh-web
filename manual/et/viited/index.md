# Viiteosa

Viiteosa vastab küsimusele „millised väärtused on lubatud ja mida see väli
teeb?”. Õpetuslikud näited asuvad [Õppimise](/et/oppimine/) ning praktilised
protsessid [Töövoogude](/et/toovood/) all.

| Teema | Sisu |
|---|---|
| [Cue-tüübid](/et/viited/cue-tuubid) | Kõik cue'd, nende target ja kestuse tähendus |
| [Action'id ja triggerid](/et/viited/actionid) | Action-väljad, triggerid, käsud ja kurvid |
| [Parameetrid](/et/viited/parameetrid) | Opacity, volume, pan ning aadressid |
| [Klaviatuuri otseteed](/et/viited/otseteed) | Editori kiirklahvid |
| [Projektifail](/et/viited/projektifail) | `.imsn` struktuur ja turvaline muutmine |
| [Mõisted](/et/viited/moisted) | Showmeshi terminoloogia |

commit'i `9cc2347` põhjal.
:::
