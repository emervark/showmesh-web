# Mõisted

| Mõiste | Selgitus |
|---|---|
| **Action** | Cue sees olev käsk või parameetrimuutus |
| **Arming / standby** | Cue ettevalmistamine kiireks käivituseks |
| **Cue** | Üks käivitatav show-sündmus |
| **Cue-list** | Cue'de järjestatud loend |
| **Editor** | Electron/React kasutajaliides |
| **Engine** | C++ protsess, mis hoiab olekut ja esitab show'd |
| **Follow** | Järgmise cue automaatne käivitus pärast eelmise lõppu |
| **GO** | Käivita playhead'i järgmine cue |
| **Headless** | Mootori töö ilma editorita |
| **Layer** | Visuaalse cue z-järjekorra arvuline tase |
| **Mapper** | Reegel, mis teisendab transition'i väärtuse tajuliseks või füüsiliseks domeeniks |
| **NDI** | Võrgupõhine audio/video transpordiprotokoll |
| **PANIC** | Kõigi cue'de kohene kõva peatamine |
| **Playhead** | Cue-listi asukoht, mille järgmise rea GO käivitab |
| **Pre-flight** | Kontroll enne show'd: meedia, väljundid ja ühendused |
| **Program** | Publikule või järgmisele süsteemile saadetav lõplik komposiit |
| **Resource** | Fail või tekstiallikas, millele cue viitab |
| **Show mode** | Edit-lock režiim live-kasutuseks |
| **Trigger** | Action'i käivitav elutsüklihetk või väline sisend |
| **Trim** | Cue-tasandi parameeter, mis rakendub ressursitaseme kõrval |
| **Transition** | Parameetri muutus aja jooksul |
