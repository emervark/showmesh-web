## Source version

This first English edition documents the Showmesh application as found on the
GitHub branch `claude/transitions-v2-o0jbi1` on **15 July 2026**, commit
`9cc234700f054ae696843b63573c40cc48aeef5a`. The source code and parts of the
interface still use the product name **Intermission**; this manual uses the
user-facing name **Showmesh**.

::: warning Development version
Showmesh is under active development. Pages distinguish working features,
partial features, and future plans. If the UI and manual disagree, check the
[content status](/manuaal/sisu-olek).
:::

## How to use this manual

- New users should begin with [Getting started](/alustamine/).
- Experienced cue-based show operators can go directly to [Workflows](/toovood/).
- Use search or the [Reference](/viited/) for a field or cue type.
- Developers and writers should read [Contributing to the manual](/manuaal/kaastoo).
